# flask-V2EX仿站

###### 这是一个使用flask框架编写的论坛类型网站



#### 功能

------



- 注册/登录系统

- 个人信息设置、头像上传

- 用户动态展示

- 主题分类、发布、修改、删除

- 评论、收藏提醒

- 后台管理

#### 如何使用

------

```
使用pip安转所需依赖库；
进入flask shell，执行flask forge命令创建数据库，生成虚拟数据；
启动 run.py运行。

```
